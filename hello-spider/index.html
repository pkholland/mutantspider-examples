The mouse tracking and updating are implemented using simple C++ in the hello_spider.cpp file. See the Paint method for the code that generates the black dot, and the HandleInputEvent method for tracking the mouse position. On every mouse event the mouse_loc_x_ and mouse_loc_y_ member variables are upated from the event information passed in by the browser. Then a call to Paint is issued. Paint walks all pixels, computing the distance from that pixel to the point <mouse_loc_x_, mouse_loc_y_>, clipping this distance value to no more than 255. The red component of the pixel in question is then set to this distance, and the other components (green and blue) are set to 0. Once the entire bitmap is painted this way it issues a call to display the bitmap.
